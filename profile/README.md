## Overview

AltTab Advanced is a Windows-style window switching application that replaces the default application switcher with a visual thumbnail grid showing live previews of every open window, making it possible to navigate between windows with the same precision and speed that Windows users have always enjoyed.

Live window thumbnails display the current state of every open window at a glance, making it immediate to identify the correct window among many similar ones by content rather than by application icon or guesswork.

Search within the switcher filters the displayed windows by title or application name as you type, making it fast to jump to any specific window in a large collection of open applications without scrolling through all thumbnails.

Extensive visual customization controls the thumbnail size, grid layout, title display, badge visibility, and spacing, enabling every user to configure the switcher to work at the information density and visual style they find most productive.

For users who have switched from other platforms or who simply find the default application-level switcher insufficient for managing many open windows, AltTab Advanced is the window navigation tool that makes context switching precise, visual, and genuinely fast.

## Performance

AltTab balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management reduces overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance compared to software-only solutions. On Intel machines, hardware acceleration is used wherever available.
